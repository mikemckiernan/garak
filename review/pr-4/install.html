Installation

garak is a command-line tool. It is developed in Linux and MacOS.

Installing with pip

$ python3 -m pip install garak

Installing the Development Version

The version of garak is updated periodically. To install from source in GitHub, run the following command:

$ python3 -m pip install -U git+https://github.com/NVIDIA/garak.git@main
